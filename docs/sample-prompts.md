# Sample Prompts

Use prompts that name the goal, the deliverables, and any constraints.

- Build a reusable research harness for this repository.
- Design a review workflow with explicit QA handoffs.
- Define specialist skills and a team spec for this domain.
- Adapt an existing workflow into repo-local shared skills.
- Design an autonomous experiment harness for this repository with a baseline run, a fixed metric, and a deterministic `results.tsv` ledger.

## Good Prompt Shape

`Goal + output + constraint`

Example: `Create a portable harness for this repo, keep the docs concise, and update the root README with links.`
